# 📧 Email Spam Detection System

An intelligent Machine Learning project that detects whether an email or message is **Spam** or **Not Spam** using Natural Language Processing (NLP) and the Naive Bayes algorithm.

---

## 🚀 Project Overview

This project uses Machine Learning techniques to analyze text messages and classify them as spam or ham (not spam).
It includes text preprocessing, feature extraction using TF-IDF, model training, and a Flask web application for real-time predictions.
